Project notes
What counts as contribution?

The current dataset counts merged pull requests, grouped by contributor, organization, and project.

Why crypto ecosystems?

Crypto infrastructure is built in public across many repositories, so merged open-source work is a useful signal for contributor discovery.

Is the project open source?

Yes. The dashboard and scripts live in the public repository linked in the header.

Can the methodology evolve?

Yes. Future versions can add richer signals, but the baseline stays simple: visible work merged into public repositories.

Why merged pull requests?

Merged pull requests are easy to audit, reproducible through GitHub data, and strongly tied to accepted work in a project.

Where is the source code?

The source is available from the Repository link in the header.
