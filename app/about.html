Open-source crypto ecosystem project

A public contribution map for crypto builders.

Proof of Contribution pulls public contribution data directly from GitHub and turns merged pull requests into a simple contributor layer: who builds, where they contribute, and which projects concentrate open-source activity.

Why it exists

Crypto ecosystems rely on public infrastructure, but contribution is scattered across many repositories. This project makes that work easier to inspect.

What it measures

The current version focuses on merged pull requests by contributor, organization, project, and date. It is intentionally transparent and easy to reproduce.

Where data comes from

The dataset is extracted from GitHub's public API when the project is refreshed, so the dashboard reflects the latest available repository activity from the scanned organizations.

Why it is useful

Raw PR data becomes more valuable once it is grouped into metrics: contributors, projects, organizations, and contribution concentration can be compared from the same public source of truth.
